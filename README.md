# grove-docgen

LLM-powered, workspace-aware documentation generator.

## Installation

```bash
grove install docgen
```

## Usage

```bash
docgen --help
```

## Development

### Building

```bash
make build
```

### Testing

```bash
make test
make test-e2e
```

### Linting

```bash
make lint
```
